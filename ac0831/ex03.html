이벤트

웹 브라우저나 사용자가 행하는 어떠한 동작을 의미

JS 는 이러한 이벤트 기반의 처리를 위한 언어

1. 마우스 이벤트
속성 설명
click 요소를 클릭시 발생하는 이벤트
dbclick 요소를 더블 클릭시 발생하는 이벤트
mousedown 요소를 누르는 동안 발생
mouseup 요소를 땔 때 발생
mouseover 요소 위로 마우스가 옮겨질 때 발생
mouseout 요소를 벗어날 때 발생

2. 키보드 이벤트
속성 설명
keypress 키보드 키를 눌렀을 때
keydown 키를 계속 누르고 있을 때
keyup 키를 땟을 때

3. 문서 이벤트
속성 설명
abort 페이지가 완전히 로딩되기전 취소 했을 때
error 페이지가 정확히 로딩되지 않을 때
load 페이지 로딩이 끝났을 때
resize 페이지 화면 크기가 변경시
scroll 페이지가 스크롤될 때
unload 페이지를 벗어날 때

4. 홈 이벤트
속성 설명
blur 폼 요소에서 커서가 빠져 나올 때
change 목록이나 체크 상태가 변경되었을 때
focus 폼 요소에 커서가 놓였을 때 발생
reset 폼이 다시 시작 되었을 때
submit submit 버튼을 눌렀을 때 발생
mouseout 요소를 벗어날 때 발생
