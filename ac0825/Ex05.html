시멘틱(semantic) 태그

의미를 가지는 태그

  • 직접적인 효과가 있는 태그는 아니다
  • 웹 브라우저 및 검색 엔진이 찾기 좋은 태그
  • HTML5에서 지원하고 이전 버전에서 보통 div로 사용했다

종류

  1. header : 페이지의 상단부에 사용
  2. main : 주 컨텐츠에 사용
  3. section : 주제 별로 나눌 수 있는 영역 ! 주로 main 안에서 사용 됨
  4. article : 개별 컨텐츠를 나타내는 영역
  5. aside : 좌/우측에 사이드 메뉴 영역
  6. nav : 네비게이션이라는 의미로 주 메뉴 영역
  7. footer : 페이지 하단부에 사용
